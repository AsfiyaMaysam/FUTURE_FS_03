# Blue Haven — Luxury Ocean-Inspired Fine Dining Website
✨ Premium Fine Dining Experience • Modern Ocean Theme • Fully Responsive ✨

## 🎯 Project Overview

**Blue Haven** is a modern luxury fine dining restaurant website with an elegant ocean-inspired theme.  
It was created as part of a full-stack web development learning/project task (2026).

The project focuses on:

- Sophisticated & immersive user interface  
- Smooth scrolling & interactive experience  
- Mobile-first responsive design  
- Real high-end restaurant feel  

## 🚀 Live Demo

🔗 [Visit Blue Haven ](https://asfiyamaysam.github.io/FUTURE_FS_03/)

## ✨ Features

### 🧭 Core Sections

- 🏠 Hero section with gradient ocean overlay  
- 🌊 About the restaurant  
- 🍽️ Interactive Menu with category filters  
- 🖼️ Gallery with hover zoom effect  
- ⭐ Customer testimonials carousel  
- 📍 Contact section + embedded Google Map  
- 📩 Reservation form (desktop + modal)  

### 🎨 Premium UI Enhancements

- ✅ Sticky glassmorphism navbar  
- ✅ Active navigation link highlight  
- ✅ Smooth scroll behavior  
- ✅ Mobile-friendly hamburger menu  
- ✅ Dark / Light mode toggle (sun/moon icon)  
- ✅ Responsive layout (desktop • tablet • mobile)  
- ✅ Floating reservation button (optional future)  
- ✅ Year auto-update in footer  

## 🛠️ Tech Stack

| Technology     | Purpose                          |
